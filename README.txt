📦 Plantilla Web de Boda - Versión 1.0

📂 Estructura:
- index.html       → Contenido principal
- style.css        → Estilos visuales
- script.js        → Funcionalidad de cuenta regresiva
- imagenes/        → Carpeta con fotos de ejemplo

🔧 Pasos para usar:
1. Guarda todos los archivos como se muestra arriba.
2. Reemplaza las fotos en la carpeta "imagenes/" por las tuyas.
3. Edita index.html para personalizar los textos.
4. En script.js cambia la fecha por la de tu boda.
5. Abre index.html en cualquier navegador.

📷 Fotos incluidas:
- foto1.jpg, foto2.jpg, foto3.jpg → ejemplos para la galería
- fondo-agua.jpg → imagen suave para fondo

💡 Consejo: Si quieres publicarlo online, puedes usar GitHub Pages, Netlify o Vercel.
